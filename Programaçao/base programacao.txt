EXPLICACAO DO CONCEITO VARIAVEL: VARIAVEIS LUGAR ONDE GUARDA OS DADOS

                   VARIAVEIS 

{ INICIO
  EXIBIR
  PROCESSAR
  FIM
} 


                 TIPOS DE VARIAVEIS 
{
 * TIPO INTEIRO (RECEBE NUMEROS POSITIVOS E NEGATIVOS QUE NAO SEJAM DECIMAIS )

 * REAL ( RECEBEM NUMEROS POSITIVOS E NEGATIVOS MAS QQUE PODEM CONTER CASAS DECIMAIS)

 * TEXTO (GUARDA INFORMACOES DO TIPO TEXTO)
 
 * LOGICO (RECEBE O QUE E VERDADEIRO OU FALSO )

 * TEMOS TAMBEM A ENTRADA MANUAL QUE RECEBE DADOS OU INFORMACOES 
}

                  OPERAORES ARTIMETRICOS 
{
 * OPERADOR /OPERACAO
   +         SOMA 
   -         SUBTRACAO          
   *         MULTIPLICACAO        
   /         DIVISAO
   MOD       RESTO DA DIVISAO

EX: 9/2 = 4,5 VARIAVEL REAL
    9/2 = 4 VARIAVEL INTEIRO
    9MOD2= 1

NESSE EX USAMOS O MOD PARA DIZER QUANTO FALTAVA PARA O RESULTADO FINAL NA VARIAVEL INTEIRO

OBS: SE O RESULTADO DO CALCULO DE UM NUMERO MOD2 FOR IGUAL A ZERO, SIGNIFICA QUE O NUMERO E PAR

}
      
                        OPERADORES RELACIONAIS 

{
  IGUAL A (==)

  EX: 5==5 VERDADEIRO
      8==4 FALSO

  MAIOR QUE (>)

  EX: 4>8 FALSO
      8>4 VERDADEIRO
 
  MENOR QUE (<)
  
  EX: 4<2 FALSO
      2<4 VERDADEIRO

  MAIOR OU IGUAL A (>=)

  EX: 8>=4 VERDADEIRO
      4>=4 VERDADEIRO
      4>=8 FALSO
 
  MENOR OU IGUAL (<=)
 
  EX: 8<=4 FALSO
      4<=4 VERDADEIRO

  DIFERENTE CE (!=)

  EX: 8!=4 VERDADEIRO
}

                        OPERADORES LOGICOS


{

    E(AND) (&&) SO RESULUTA EM VERDADEIRO SE TODOS OS VALORES FOREM VERDADEIROS

VALOR 1     VALOR 2        RESULTADO

VERDADEIRO  VERDADEIRO   VERDADEIRO
VERDADEIRO  FALSO        FALSO
FALSO       VERDADEIRO   FALSO
FALSO       FALSO        FALSO
 
      OU(OR) (I I)

VALOR 1            VALOR 2     RESULTADO
 
VERDADEIRO         VERDADEIRO  VERDADEIRO
VERDADEIRO         FALSO       VERDADEIRO
FALSO              VERDADEIRO  VERDADEIRO
FALSO              FALSO       FALSO

       NAO (NOT)(!)

INVERTER O VALOR

VALOR 1     RESULTADO

VERDADEIRO  FALSO
FALSO       VERDADEIRO
