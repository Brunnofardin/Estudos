Trabalhando com media queries e media features

Tipos de media queries

Meida All, media Screen, media Print

Exitem diversos tipos de media queries mas os mais usados são:

  • Media ALL
  • Media Screen
  • Media Print

Media Print

Media print serve para impressoras, não tem quando você dá ctrl+p em site para imprimi-lo? Já paraou pra pensar que aquela é uma versão diferente do site? Que o menu some,os links somem? Pois é meu amigo isso é uma versão diferente do site, usada com a media querie do tipo print,que traduzido para o portgugûes significa "IMPRIMIR"

Media Sreen

Media screen serve para telas sem especificar o tamanho digamos que é uma versão que atende a todos mais que naverdade causa vários problemas, citados abaixo:

  • Responsividade
  • Tamnho inadequados

Media ALL

Como o própio nome diz ('TODOS') será uma vesão para todos tipos de tela que não ira mudar de modo algun
