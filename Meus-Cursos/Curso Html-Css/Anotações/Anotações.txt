PARA TORNAR UMA IMAGEM FLEXVEL USAMOS 2 COMANDOS:

MIN-WIDTH:200PX      - LARGURA MÍNIMA QUE UM ELEMNTO DEVE TER

MAX-WIDTH:1040PX     - LARGURA MÁXIMA QUE UM ELEMENTO PODE CHEGAR

ALGO IMPORTANTE DE CITAR É QUE, A IMAGEM SE ADAPTA AO TAMNHO DA TELA ENTRE 200PX A 1040PX COMO FOI USADO NO EXEMPLO.




---- MODULO-3 TRABALHANDO COM IMAGENS----

BACKGROUND-IMAGE : URL(LOCAL DO ARQUIVO) - CARREGA UMA IMAGEM DE FUNDO.

BACKGROUND-IMAGE:REPEAT; - SERVE PARA FAZER UMA IMAGEM SE REPETIR VÁRIAS VEZES NO FUNDO DO TEU SITE.

BACKGROUND-SIZE: PX PX; - SERVE PARA DEFINIR A ALTURA E LARGURA DA SUA IMAGEM DE FUNDO.

-------- TARBALHANDO OM TABELAS--------


PARA CIRAR UMA TABELA USAMOS A TAG <TABLE>


A ESTRUTURA DE UMA TABELA BÁSICA É :  TABLE  - TABELA
                                          TR - LINHA
                                            TD - DADO


VERTICAL-AILING - DECIDE ONDE SEU TD - DADOS  IRAM FICAR SEJA NO TOPO OU EMBAIXO ETC.. DE FORMA VERTCAL




TABELAS GRANDES :



          THEAD - CABEÇALHO DA TABELA 
          TBODY - CORPO DA TABELA
          TFOOT - RODAPÉ DA TABELA


ABREVIAÇÕES

TH - ALGO IMPORTANTE

TD - UM DADO 

TR -  UMA LINHA

COLSPAN - ESPANSÃO DE COLUNA

COLGROUP
